# Screenshots Directory

This directory contains screenshots of the Link Fetch Chrome extension for documentation purposes.

## Required Screenshots

To complete the README, please add the following screenshots:

1. `popup.png` - Screenshot of the main popup interface showing collected links
2. `sidepanel.png` - Screenshot of the side panel interface

## How to Take Screenshots

1. Install the extension in Chrome
2. Open a webpage with multiple links
3. Click the Link Fetch icon to open the popup
4. Use the browser's screenshot tool or an external screenshot tool to capture the popup
5. For the side panel, open the side panel and capture it separately

## Tips for Good Screenshots

- Make sure the extension shows some collected links to demonstrate functionality
- Capture both light and dark mode if possible
- Ensure the screenshots are clear and high-resolution
- Crop screenshots to focus on the extension UI without unnecessary browser elements
